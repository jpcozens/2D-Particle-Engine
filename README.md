# 2D-Particle-Engine

**An interactive demonstration environment to simulate 2D particle physics and collisions. Designed with modular architecture for future external integration in pygame projects.**

This project uses ideas from the agile software development methodology, for more information on the development process please visit the [project overview](docs/OVERVIEW.md).

---

## Installation & Dependencies

This project uses pygame for rendering. In order to run the program, you need to ensure you have it installed.

```sh
pip install pygame
```
> When using the windows installer for python, pip (and python) will only be added to PATH if specifically told to.

<br>

To install the project, simply clone to repository to your system at your desired directory.
```sh
git clone https://github.com/jpcozens/2D-Particle-Engine
```

<br>

Finally, to launch the program, you can simply run the main file from the src directory.
```sh
cd 2D-Particle-Engine/src
python main.py
```

**OR**

You can navigate to the main.py file in your file navigation system and launch the file manually.
